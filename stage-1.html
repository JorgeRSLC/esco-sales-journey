Initial ask.

ESCO asks the prospect if they would like to schedule an appointment for a free evaluation and proposal.

What happens

Every prospect, regardless of size or vertical, begins here with the same question. The ask is simple, low-commitment, and free — designed to gather an initial signal rather than to close.

What we're listening for

The response to this single ask sends the prospect down one of two lanes. The answer itself is a signal, and how quickly they want the walkthrough scheduled is a second signal layered on top.

Where it goes next

  • Yes → Stage 02 · Accepts evaluation. Enters the proposal lane. At least semi-critical.
  • No → Stage 02 · Declines evaluation. Enters the nurture lane. Most likely non-critical.

Open questions

  • Do we want to track the channel of the initial ask (phone, email, in-person, referral) as a separate state signal?
  • Should there be a scripted follow-up probe if the answer is ambiguous ("maybe later")?

State Signals

No state assigned at this stage. State is first signaled at Stage 02.

Timing

Ask cadence Single ask
Response target Same contact

Outputs

  • If Yes Proceeds to Stage 02 · Accepts evaluation
  • If No Proceeds to Stage 02 · Declines, then Stage 02a
